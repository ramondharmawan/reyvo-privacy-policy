# Kebijakan Privasi Reyvo

**Terakhir diperbarui:** [TANGGAL BERLAKU, mis. 9 Juni 2026]

Kebijakan Privasi ini menjelaskan bagaimana **[NAMA DEVELOPER/PERUSAHAAN]** ("kami")
mengumpulkan, menggunakan, dan melindungi data Anda saat menggunakan aplikasi
**Reyvo** ("Aplikasi"). Dengan menggunakan Aplikasi, Anda menyetujui praktik yang
dijelaskan di dokumen ini.

Jika Anda memiliki pertanyaan, hubungi kami di **[EMAIL_SUPPORT, mis. support@reyvo.app]**.

---

## 1. Ringkasan Singkat

- Reyvo memiliki **mode gratis yang berjalan sepenuhnya di perangkat Anda** (offline).
  Catatan dan transkripsi suara di mode gratis **disimpan lokal di perangkat**, tidak
  dikirim ke server kami.
- Fitur **Pro (berbayar)** bersifat opsional dan memerlukan akun. Fitur Pro
  mengirim sebagian data (mis. rekaman suara untuk transkripsi, catatan untuk
  sinkronisasi) ke layanan cloud kami untuk diproses.
- Pembayaran langganan diproses oleh **Google Play**. Kami **tidak pernah** menerima
  atau menyimpan nomor kartu/instrumen pembayaran Anda.

---

## 2. Data yang Kami Kumpulkan

### 2.1 Mode Gratis (tanpa akun)
- **Catatan & transkripsi suara**: dibuat dan disimpan **secara lokal** di perangkat
  Anda. Kami tidak dapat mengaksesnya.
- **Izin mikrofon**: digunakan untuk fitur ucapan-ke-teks (speech-to-text) yang
  diproses **di perangkat**. Audio mode gratis tidak diunggah ke server kami.

### 2.2 Fitur Pro (memerlukan akun)
- **Data akun**: alamat email, dan (jika login dengan Google) nama serta ID akun
  Google Anda. Digunakan untuk autentikasi.
- **Rekaman suara**: untuk transkripsi panjang berbasis AI, rekaman audio diunggah ke
  penyimpanan cloud kami, diproses, lalu dapat dihapus sesuai jadwal retensi
  (lihat Bagian 5).
- **Konten catatan tersinkronisasi**: transkrip, ringkasan, daftar tugas, dan jadwal
  yang Anda pilih untuk disinkronkan ke cloud.
- **Data Google Calendar** (jika Anda mengaktifkan sinkronisasi): token otorisasi
  untuk membuat/memperbarui acara di kalender Anda.
- **Data langganan**: status langganan, tanggal perpanjangan/berakhir, dan token
  pembelian dari Google Play (bukan data kartu Anda).

### 2.3 Data Teknis
- Informasi dasar yang diperlukan agar layanan berfungsi (mis. log error,
  identifier permintaan). Kami berusaha meminimalkan data ini.

---

## 3. Bagaimana Kami Menggunakan Data

Kami menggunakan data Anda untuk:
- Menyediakan dan mengoperasikan fitur Aplikasi (transkripsi, ringkasan, sinkronisasi).
- Memproses dan mengelola langganan Pro.
- Memproses rekaman suara menjadi teks dan mengklasifikasikannya (catatan/tugas/jadwal)
  menggunakan penyedia AI.
- Menyinkronkan tugas/jadwal ke Google Calendar Anda (jika diaktifkan).
- Menjaga keamanan, mencegah penyalahgunaan, dan memenuhi kewajiban hukum.

Kami **tidak menjual** data pribadi Anda.

---

## 4. Pihak Ketiga / Subprosesor

Untuk menjalankan fitur tertentu, data dapat diproses oleh penyedia berikut:

| Penyedia | Tujuan | Data terkait |
|---|---|---|
| **Google Play Billing** | Memproses pembayaran langganan | Status & token pembelian (bukan data kartu) |
| **InsForge** (backend/hosting, region Asia Tenggara) | Autentikasi, penyimpanan cloud, pemrosesan server | Email, catatan tersinkron, rekaman Pro |
| **Google Gemini API** | Transkripsi & klasifikasi AI | Rekaman/transkrip yang diproses |
| **Google Calendar API** | Sinkronisasi tugas/jadwal | Acara kalender yang Anda buat lewat Aplikasi |

Masing-masing penyedia memiliki kebijakan privasinya sendiri.

---

## 5. Penyimpanan & Retensi Data

- **Data mode gratis**: tetap di perangkat Anda; terhapus saat Anda menghapus catatan
  atau meng-uninstall Aplikasi.
- **Rekaman suara Pro**: disimpan sementara untuk transkripsi lalu dihapus otomatis
  setelah **[mis. 7 hari]**.
- **Catatan cloud & data langganan**: disimpan selama akun Anda aktif.
- Anda dapat meminta penghapusan akun dan data terkait kapan saja (lihat Bagian 7).

---

## 6. Keamanan

- Token autentikasi disimpan terenkripsi di perangkat (Keychain/penyimpanan aman OS).
- Akses data di server dibatasi per pengguna (Row-Level Security): pengguna hanya
  dapat mengakses datanya sendiri.
- Kunci API penyedia AI disimpan sebagai secret server-side, tidak pernah ada di
  aplikasi klien.

Tidak ada sistem yang 100% aman, namun kami menerapkan langkah-langkah yang wajar
untuk melindungi data Anda.

---

## 7. Hak Anda

Anda berhak untuk:
- Mengakses, memperbaiki, atau menghapus data pribadi Anda.
- Menghapus akun Pro Anda beserta data cloud terkait.
- Mencabut izin Google Calendar kapan saja melalui pengaturan akun Google Anda.
- Mengelola atau membatalkan langganan melalui **Google Play → Langganan**.

Untuk permintaan terkait data, hubungi **[EMAIL_SUPPORT]**. Kami akan menanggapi dalam
waktu yang wajar sesuai hukum yang berlaku.
